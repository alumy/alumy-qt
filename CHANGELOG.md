## [0.0.1] - 2019-01-15

### Added

- Initial release
- CRC module: CRC16, CRC32, Modbus CRC16 checksum algorithms
- Audio module: WAV audio file read/write operations
- Network module: SLIP protocol implementation
- Memory module: Memory management utilities
- String module: String handling functions (strlcpy, etc.)
- CSV module: CSV file parsing based on libcsv
- Protobuf-C module: Protocol Buffers C implementation
- XYZModem module: YModem file transfer protocol
- Log module: Logging system with spdlog integration
- Singleton pattern support
- CMake build system with cross-compilation support
- Qt5/Qt6 compatibility
